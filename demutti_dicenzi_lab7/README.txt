ImageMatching and ImageRetrival folders are not included in order for the overall folder not to exceed the maximum dimensions.
In order for everything to work properly, 'Optional' folder has to be copied and pasted in the same 'main' folder that contains
ImageMatching and ImageRetrieval folders.
'Optional' contains the optional code that allows to perform image retrieval using the number of detected matches:
it is sufficient to run file 'Labo8_part3.m' to see the final result.
